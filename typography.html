Text level elements

Text levelHTML elements may be used within other elements. They include:

  • <a href="#">link or anchor</a>
  • <abbr> represents an abbreviation or acronym
  • <em> for emphasized text
  • <strong> for strongly emphasized
  • <dfn> for definition of a term
  • <code> for code fragment
  • <samp> for sample text
  • <kbd> for keyboard input
  • <var> for variable
  • <cite> for short citation
  • <i> for italics
  • <b> for bold
  • <u> for underlined text
  • <s>for text that is no longer correct
  • <big> for larger text
  • <small> for smaller text
  • <sup> for superscript
  • <sub> for subscript
  • <del> for visibly deleted
  • <ins> for inserted content
  • <dfn> for definitions
  • <mark> for highlighted passages
  • <hr> for horizontal line
